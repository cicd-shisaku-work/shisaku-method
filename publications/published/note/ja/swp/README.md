# SWP: シサク書き手原則フレーム（Shisaku Writer's Principles）

シサク個人の「書き手としてのこだわり」を、機構つきの原則として言語化した認知フレームと、その運用文書群。シサク認知フレーム理論のモード(I)適用例。

**これは良い文章の一般則ではない。** シサクという一人の書き手の原則である。他者にとっての価値は、中身でなく形式——「自分のこだわりを、機構つきの原則として言語化する」という方法の見本——にある。

## ファイル構成

| ファイル | 内容 | 読者 |
|---|---|---|
| `swp-writer-principles.md` | フレーム本体。中心の定義、土台、四系統十二軸、優先順位、境界、倫理、更新履歴 | 人間・AI |
| `swp-claude-generation-guideline.md` | 原稿生成前の対話手順（代弁の前に、質問）。AIがシサクの一人称原稿を書く際に、SWPとあわせて装着する | AI（Claude） |
| `swp-voice-extraction-prompts.md` | 読者配布用。自分の声を法則化するためのプロンプト集と運用条件 | 人間（一般読者） |

## 層の分離

- **原則層**（writer-principles）：どの記事でも守られる不変のこだわり。変わりにくい
- **手順層**（generation-guideline）：AIとの協働の手順。運用で更新される
- **配布層**（extraction-prompts）：読者が自分用に同じことをやるための道具。最も短命

上位の変更は下位に波及する。下位の変更は上位に波及しない（ただし手順層・配布層での発見が、反証として原則層を更新することはある——更新履歴を参照）。
